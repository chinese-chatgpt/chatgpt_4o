4. **开始对话**：享受中文交流，支持翻译、写作、办公等智能服务。

### 方法二：通过官网使用 ChatGPT

1. **访问官网**：前往 [ChatGPT 官网](https://chat.openai.com)（需翻墙）。
2. **注册账号**：
   - 准备一个有效的海外手机号进行验证。
   - 输入邮箱和密码完成注册。
3. **选择版本**：登录后选择 GPT-4。
4. **进行对话**：输入问题，体验 OpenAI 的智能解答。
